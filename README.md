## Overview
This project is an AI-powered tool designed to help students study more efficiently by summarizing notes, generating quizzes, and providing explanations.

## Features
- Note summarization
- Quiz generation
- Flashcards
- Interactive chatbot

## Technologies Used
- Python
- (later: OpenAI API or similar)
